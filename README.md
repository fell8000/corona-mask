#### <서울 공적 마스크 판매처 지도>

<img src="https://user-images.githubusercontent.com/63782897/91561382-736a9900-e976-11ea-80e2-02984a190dcc.PNG" width="90%"></img>

마커 클러스터 이용, 멀리 이동하면 여러개가 하나로 합쳐짐

#### <사용 API>

링크 : https://www.data.go.kr/data/15057411/openapi.do

#### <사용 문법>

+ folium
+ pandas
